## Getting Started

```
git clone https://github.com/njbrown/texturelab.git

# if you want to pull down assets (textures and node icons)
git submodule update --init

cd texturelab
```

yarn:

```
yarn install
yarn electron:serve
```

npm:

```
npm install
npm run electron:serve
```
